Análise de Faturamento – Base AdventureWorks

Contexto
Neste projeto utilizei a base AdventureWorks para responder algumas perguntas simples de negócio relacionadas a faturamento e comportamento de clientes.

O objetivo foi praticar:
•	GROUP BY
•	HAVING
•	Subqueries
•	TOP
•	Cálculo percentual

Mas principalmente transformar consulta técnica em visão de negócio.

Perguntas respondidas
1.	Quem são os 10 clientes que mais faturaram?
2.	Qual o ticket médio por cliente?
3.	Qual o faturamento total da base?
4.	Quanto os Top 10 representam (%) do total?
5.	Quantos clientes fizeram apenas 1 pedido?

O que eu quis analisar

A principal ideia foi entender:
•	Se a receita é concentrada em poucos clientes
•	Se existe dependência de grandes contas
•	Se a base é pulverizada
•	Como se comporta o ticket médio
